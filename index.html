My First Custom-Coded Website Theme 🎉

This is a simple site built on HTML and CSS. I designed it with a few key goals in mind: speed, responsiveness, and to gain experience using intermediate-to-advanced CSS.

Ultimately, I plan to establish myself as a go-to web designer for small businesses in my area. I believe there is a strong market for local businesses who would find a subscription model attractive.

Key Features 💻

  • Modular Code Structure - I've organized code by feature so that it's extremely easy to make changes intentionally and confidently.

  • Lighting Fast & Secure - Since this site is built using only HTML and CSS, it loads instantly and is nearly impossible to hack.

  • Responsive to All Screen Sizes - I spent hours carefully constructing media queries so that this site automatically adjusts to the size of your screen.

  • Functional Blog System - Creating blog pages is manual on this site, but I'd argue that its a good thing from a quality-control perspective.
